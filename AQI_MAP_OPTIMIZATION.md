# AQI 地圖優化說明

## 優化內容

### 1. 簡化分色顯示
採用三分色方案，更直觀地顯示空氣品質：

- **綠色 (#00E400)**: AQI 0-50 - 良好
- **黃色 (#FFFF00)**: AQI 51-100 - 普通  
- **紅色 (#FF0000)**: AQI 101+ - 不健康

### 2. 優化資訊視窗
點擊測站標記顯示簡潔資訊：

- 測站名稱（醒目顯示）
- 所在地（縣市）
- 即時 AQI 數值（突出顯示）
- 空氣品質狀態
- 更新時間

### 3. 動態標記大小
根據 AQI 數值調整標記大小：

- AQI ≤ 50: 小標記 (radius=6)
- AQI 51-100: 中標記 (radius=8)  
- AQI 101+: 大標記 (radius=10)

### 4. 改進地圖設計
- 新增地圖標題：「台灣空氣品質指標即時地圖」
- 優化圖例設計，使用圓形色塊
- 半透明背景，提升視覺效果
- 更好的字體和間距

## 使用方法

### 快速測試
```bash
cd scripts
python test_aqi_map.py
```

### 完整分析（需要 EPA API 金鑰）
1. 設定環境變數：
   ```env
   EPA_API_KEY=your_api_key_here
   ```

2. 運行完整分析：
   ```bash
   python run_aqi_analysis.py
   ```

3. 在瀏覽器中打開生成的地圖檔案

## 輸出檔案

- **地圖檔案**: `output/maps/aqi_map_YYYYMMDD_HHMMSS.html`
- **數據檔案**: `data/processed/aqi_data_YYYYMMDD_HHMMSS.csv`
- **分析報告**: `output/reports/aqi_report_YYYYMMDD_HHMMSS.txt`

## 技術特點

### 顏色編碼
使用標準 EPA 顏色：
- 綠色：符合標準的「良好」顏色
- 黃色：標準的「普通」顏色
- 紅色：標準的「不健康」顏色

### 響應式設計
- 資訊視窗自適應內容長度
- 圖例和標題固定位置，不受地圖縮放影響
- 工具提示簡潔清晰

### 數據處理
- 自動過滤無效座標
- 數值類型轉換和錯誤處理
- 缺失值處理

## 注意事項

1. **API 限制**: 請遵守 EPA API 使用規範
2. **網路連接**: 需要穩定的網路連接獲取即時數據
3. **瀏覽器支援**: 建議使用現代瀏覽器（Chrome, Firefox, Safari）
4. **座標系統**: 使用 WGS84 (EPSG:4326) 座標系統
